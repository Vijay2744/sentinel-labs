# 🧪 Sentinel Labs

## Overview

Building an intelligent decision-support system using AI APIs to simplify complex financial understanding and automate responses.

---

## Why This Matters

Financial products are often confusing:

* Eligibility unclear
* Limits hard to understand
* Processes complex
* Users need faster answers

---

## What Sentinel Solves

Helps users understand:

* Product eligibility
* Loan limits
* Required next steps
* Financial product comparisons
* Guided responses

---

## Example Use Cases

* Am I eligible for LAS?
* How much can I borrow?
* Which loan suits me?
* Why was my request rejected?

## Current Focus

* AI API integrations
* Prompt workflows
